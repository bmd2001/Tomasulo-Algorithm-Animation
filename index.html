Welcome on my webpage, where you'll be able to see how Tomasulo's algorithm actually works. This animation is highly inspired by J.R. youtube video, where he explains every passage.

Below you will find a form that you can fill in with the Instructions and the number of clock cycles needed for executing them. When you have added everything, you can press the start button to start the animation. You will also be able to choose if you want to run it all or step through it.
